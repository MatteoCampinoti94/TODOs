# TODO for Atom

**Official GitHub page:** http://peteallport.github.io/TODO/

Introducing TODO for Atom. Organize your //TODOs and see them listed by pressing ctrl-alt-T.

### Installation
`apm install to-do`

or search for "to-do" in Atom
