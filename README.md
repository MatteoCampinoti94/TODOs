# TODOs for Atom

**Official GitHub page:** http://peteallport.github.io/TODOs/

Introducing TODOS for Atom. Organize your //TODOs and see them listed by pressing ctrl-alt-T.

### Installation
`apm install TODOs`

or search for "TODOs" in Atom
